# **DansTonChattr**.
## <u>**Catégorie**</u>

Linux

## <u>**Description**</u> :

```
Récupérer le flag contenu du fichier flag.txt.
Vous pouvez vous connecter en ssh avec l'identifiant "user1" et le mot de passe "letmein"
 
```

## <u>Hints</u> 

Chaque seconde est une nouvelle seconde et ce qui est immuable redevient immuable

## <u>Auteur</u> 

Processus Thief

## <u>Solution</u> :

user1 ne dispose pas des permissions sur le fichier flag.txt
on trouve un suid avec la commande "sudo -l"
on peut utiliser le suid chattr pour rendre les attributs du fichier modifiables
on peut ensuite modifier les permissions et lire le contenu du fichier
En revanche il y a un cron job qui rétabli les perms du fichier et le rend immuable a nouveau
il faut donc agréger les commandes :
sudo chattr -i /home/user1/flag.txt && chmod 777 flag.txt && cat flag.txt
