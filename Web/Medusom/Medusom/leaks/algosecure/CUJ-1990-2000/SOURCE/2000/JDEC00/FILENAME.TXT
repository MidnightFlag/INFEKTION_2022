Keywords: supplement, Java, semaphore, PalmOS, J2ME, performance, efficiency

The code listings for the Java Solutions supplement to the December 2000
C/C++ Users Journal include source code for Mark Nadelson's article,
"Java Named Semaphores," Peter Meehan's article, "Developing for the
PalmOS Platform using J2ME," and John Keyes article, "Increasing Java
Performance."
